# :: 원티드 프리온보딩 프론트엔드 코스 사전과제 리펙토링

## 개발환경 실행방법

\*\* node.js와 React를 구동할 수 있는 기본적인 환경이 필요합니다.

- 터미널에서 명령어 입력

```bash
// node 패키지 설치
npm install

// json 서버 실행
npx json-server ./data.json --port 4000

// clint 실행
npm run start
```

- 브라우저에서 localhost:3000 접속
